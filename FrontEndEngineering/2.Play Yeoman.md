Global install yeoman

```bash
npm install -g yo
```

Install generator to work with yeoman

```bash
npm install -g generator-node
```

Then start playing

```bash
mkdir 2playYeoman-node
yo node
```

![](./media/YeomanInitNode.png)

## Sub Generator

```
yo node:cli
```
